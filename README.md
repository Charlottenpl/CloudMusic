# CloudMusic
> MusicPlayer-Compose翻版</br>
> 目前正在开发中

## 整体思路
1. 网络请求
    - Net(https://github.com/liangjingkanji/Net) + 协程 + Flow + kotlin-serialization
2. 整体框架
    - MVI
3. 屏幕适配
    - Compose自适应
4. 数据保存
    - MMKV,Room
5. 音频播放
    - StarrySky
6. API
    - NeteaseCloudMusicApi
> 整体UI图则采用”即时设计“中的开源UI设计图 </br>
